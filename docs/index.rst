Hyperwave Community
===================

Python SDK for GPU-accelerated FDTD photonics simulation via cloud API.

Hyperwave Community provides a high-level interface for building, running, and
analyzing electromagnetic simulations on cloud GPUs. Define photonic structures
from GDS layouts, launch FDTD simulations, and visualize the results, all from
Python.
